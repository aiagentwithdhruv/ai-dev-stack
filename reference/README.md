# reference/ — imported material, pending curation

This folder holds **generic, de-identified** material consolidated from earlier
standalone repositories so nothing useful is lost. It is a **staging area**, not the
curated kit — content here will be progressively merged into the main sections
(`foundations/`, `pillars/`, `domains/`, `prompts/`) over time.

Each subfolder has a `SOURCE.md` noting where it came from and what it covers.
Anything employer-, client-, or person-specific was deliberately excluded during import.

> If you're looking for the polished, ready-to-use kit, start at the [repo root](../README.md).
